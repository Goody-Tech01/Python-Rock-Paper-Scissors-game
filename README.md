# Python-Rock-Paper-Scissors-game
## Description
This is a simple Rock-Paper-Scissors game built using Python. 
The game allows a user to play against the computer with input validation and score tracking.

## Features
1. User vs Computer gameplay
2. Random computer choices
3. Input validation
4. Score tracking system
5. Replay option

## Technologies Used
- Python

## How to Run
1. Clone or download the repository
2. Open the project in your code editor
3. Run the file:
   python rock_paper_scissors.py
